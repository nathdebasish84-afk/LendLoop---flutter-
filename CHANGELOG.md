# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-07

### Added
- Initial release of LendLoop.
- Core transaction tracking (Lend/Borrow).
- SQLite integration for local data storage.
- Dashboard with total summaries and upcoming reminders.
- Search and filtering capabilities.
- Analytics dashboard with charts (fl_chart).
- Local notification system for due date reminders.
- CSV export functionality.
- Modern UI/UX with Bottom Navigation and Animated Transitions.
- Light and Dark mode support.
- GitHub Actions CI/CD for automated APK builds.
