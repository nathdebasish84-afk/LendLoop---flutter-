# LendLoop

[![Build Android Release](https://github.com/your-username/LendLoop/actions/workflows/build_release.yml/badge.svg)](https://github.com/your-username/LendLoop/actions/workflows/build_release.yml)

**LendLoop** is a modern, intuitive Flutter application designed to help you keep track of items and money you've lent to or borrowed from others. Never lose track of your belongings or debts again.

## ✨ Features

- **Dashboard:** At-a-glance summary of total lent and borrowed amounts.
- **Transaction Tracking:** Easily log items and money with descriptions, due dates, and reminders.
- **Smart Reminders:** Get notified when a repayment or return is due.
- **Advanced Insights:** Visualize your lending habits with modern charts and analytics.
- **History & Search:** Quickly find past transactions with powerful filtering options.
- **Export to CSV:** Back up your data or share reports easily.
- **Dark Mode:** Beautifully designed themes for any time of day.

## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install) (3.0.0 or higher)
- [Android Studio](https://developer.android.com/studio) or [VS Code](https://code.visualstudio.com/)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/LendLoop.git
    cd LendLoop
    ```

2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run the app:**
    ```bash
    flutter run
    ```

## 🛠️ Built With

- **Flutter** - UI Framework
- **Provider** - State Management
- **SQLite** - Local Database
- **fl_chart** - Data Visualization
- **flutter_local_notifications** - Reminder System
